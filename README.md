# 🛰️ synth-suite: The Differentiable Biophysics Orchestrator

**synth-suite** is the meta-package for the differentiable biophysics ecosystem. it provides a unified interface to simulate and validate protein structures across multiple experimental modalities.

## 📦 Included Packages

- **synth-nmr:** Chemical shifts, RDCs, and relaxation.
- **synth-saxs:** Scattering profiles and Kratky analysis.
- **synth-afm:** Topography and tip simulation.
- **synth-cryo-em:** 3D density synthesis.
